HangFire

A simple way to kick off long-running processes from the ASP.NET request processing pipeline.

1. Install

HangFire is available as a NuGet package. To install it, type in your Package Manager Console window:

PM> Install-Package HangFire

2. Call

To call method asynchronously, call the `BackgroundJob.Enqueue` method and tell it what method needs to be invoked asynchronously.

BackgroundJob.Enqueue(() => Console.WriteLine("Hello, world!"));
